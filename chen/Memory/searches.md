# לוג חיפושים — חן

זהו הזיכרון המתמשך של חן. כל חיפוש שמתבצע מתועד כאן, כדי להימנע מחיפושים כפולים
ולעקוב אחרי המקורות שכבר נמצאו. לפני כל חיפוש — Grep על הקובץ הזה עם מילות המפתח.

פורמט ה-entry הקבוע (כל חיפוש חדש מתווסף למטה לפי התבנית הזו):

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
